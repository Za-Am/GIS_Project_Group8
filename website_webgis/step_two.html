Step 2

The purpose of this step is to produce a susceptibility map by using machine learning.
For implementing the method, the ModelMap package, implemented in R, is used that added in QGIS.


Generating a Susceptibility map will be our primer goal in the second step. so, in order to attain this map, we exploit the R programing language as an extension of QGIS and we used the Python programming language for validating the result. We implement machine learning with random Forest method (Random forests or random decision forests are machine learning methods for classification and regression and other tasks that operate by building different decision trees, decision trees themself were insufficient because they are not flexible when it comes to new data. But random forest increases the flexibility resulting in improvement of accuracy. The output of random forest is the class selected by most trees in the forest.). this methodology gives us the opportunity to generate models and validate the results and produce maps. By these maps investigating the area in detail is possible and sometimes convenient. We validate R output using a python programming language for testing points and landslide susceptibility map.


This confusion matrix shows how the RF method performed and what is the accuracy of the method. As it is evident in the matrix, the principal diagonal(100,113) indicates how many times samples were correctly classified, on contrary, in secondary diagonal(45,37), the algorithm misclassified some samples which means that where Hazard is 1, the algorithm considers it 0 and conversely. In addition, both user accuracy (UA) and producer accuracy (PA) is almost 70% so the RF method's average accuracy for our project is approximately seven out of ten.
